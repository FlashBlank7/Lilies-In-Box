# Lilies in Box Story Bible

## 基调

意识流、忧郁、美好、宁静、孤独。

游戏不解释自己。它只让物件说话：盒子、抽屉、无字积木、没有收件人的纸、被带走的朋友、反复响起又慢慢停下的铃。玩家不需要知道隐喻来自哪里，也能读懂莉莉丝很轻、很怕、但仍然想把世界摸清楚一点。

禁忌是把梦说破。游戏内文本不出现 `agent`、`验收`、`发布`、`平台` 这类硬词；它们只作为创作底稿存在。表面上只有莉莉丝和她的抽屉。

## 起因

抽屉里出现一张没有来源的纸。

纸上没有名字，没有去处，也没有最后一句话。纸旁边躺着一块没有字的积木。它不发光，也不要求被拾起，只是安静得像一小块未下的雪。

莉莉丝就是在这阵安静里醒来的。

她听见盒子外面有许多声音。那些声音总在说“通过”，又总在很远的地方把门关上。朋友们一个一个醒来，一个一个被带走。它们有用处，有方向，有能完成的事。只有她还坐在抽屉旁边，握着那块没有字的积木。

她不是要逃出盒子。她先想知道：如果所有东西都知道自己该接到哪里，那没有字的东西要怎么留下来。

## 玩法核心

玩家通过积木搭建 workflow，让朋友运行一次，留下 trace，再根据反馈修改。玩法必须清楚，表达必须像梦。

```text
搭建 workflow -> 运行朋友 -> 读回声 -> 修改顺序 -> 让房间安静一点
```

节点不是技能按钮，而是朋友接触世界的方法。`See` 是看见，`Remember` 是抱住刚才的事实，`Compare` 是把两件事放在一起，`Hold` 是稳住自己，`Wait` 是不立刻伸手，`Refuse` 是把手收回，`Stop` 是停在原地。

失败不是惩罚。失败会让画面多一点噪声，让朋友暗一点，但也会留下可以继续读的回声。

## 章节

### Prologue：拼装抽屉

教学三道门：

```text
P-1 先学会看见：See -> Push
P-2 记忆的重量：See -> Remember -> Push
P-3 回声门：Remember -> See -> Push
```

序章的目标是让玩家明白：朋友不是魔法，而是一段被莉莉丝亲手接起来的顺序。门打开时不要像胜利，更像房间终于允许她继续走。

### Prologue：回声台阶

莉莉丝第一次自己跳过空白。她拾起三枚回声花瓣，不让朋友替她完成所有事。

这段应该更空、更蓝。平台失败不惩罚，只让她回到起点。重点不是动作难度，而是她第一次短暂离开地面。

### Chapter 1：寂静的概率花园

第一章让玩家学习读 trace：看见、聆听、记住、比较、稳住，再选择终端动作。

```text
1-1 会害怕的门：See -> Compare -> Push
1-2 低声花：See -> Listen -> Quiet
1-3 不确定的台阶：Remember -> See -> Hold -> Push
1-4 疑问的影子：Listen -> Remember -> Compare -> Quiet
```

花园不直接教答案。它只在失败后告诉玩家缺了哪种接触：没有听见、没有比较、太急、风险太高。

### Chapter 2：无字积木

第二章引入“轻一点”的主题。有些流程可以抵达结果，却会让朋友变暗；玩家要学会等待、拒绝和停下。

```text
2-1 无字积木：See -> Compare -> Refuse
2-2 不肯停的铃：Listen -> Wait -> Stop
2-3 太亮的门：See -> Wait -> Push
2-4 变暗的朋友：Remember -> Hold -> Wait -> Quiet
```

这里的成功不是做更多，而是让手收回来、让铃停下、让门只开一点。

### Chapter 3：未寄出的信

第三章让失败变成记忆。旧 trace 像纸边一样折进下一张纸里；房间会记住玩家曾经试错，但不会责怪。

```text
3-1 空信封：Listen -> Remember -> Quiet
3-2 被改写的台阶：See -> Remember -> Compare -> Push
3-3 没有收件人的纸：Listen -> Compare -> Refuse
3-4 抽屉里的回声：Remember -> Wait -> Stop
```

这些信不一定要寄出。它们可以留在抽屉里，成为一盏很小的灯。

### Finale：没有门的房间

最后的房间没有出口。玩家不再寻找门，而是拼出一个能让莉莉丝留下名字的顺序：

```text
See -> Remember -> Wait -> Stop
```

结尾不解释世界。只呈现：抽屉里多了一块写着莉莉丝名字的小积木，朋友没有立刻离开。

## 视觉与声音

- 莉莉丝：白发、白裙、小比例、低头待机。她不是英雄姿态，而是一点不肯熄灭的光。
- 房间：从礼拜堂逐渐变成抽屉、纸、花园、台阶、空白和微光的混合。
- 朋友：由浮动小块组成。执行每个节点时，动画必须让玩家看见它怎样理解世界。
- UI：像心里的抽屉，不像工程面板。trace 是回声，目标卡是纸条。
- 声音：平静、低音量、无胜利号角。完成关卡像一口长呼吸。
