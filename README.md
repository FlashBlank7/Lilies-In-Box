# Lilies in Box

`Lilies` 是一个 Godot 4.7 的 2D 横版解谜叙事原型。玩家控制 AI 小姑娘莉莉丝，在一座安静的意识礼拜堂里拾取“能力积木”，把积木搭成可见、可理解的朋友。

第一章目标很小，但要完整：三个连续房间、三个积木、一个朋友、三道门。玩家先学会 `See -> Push`，再把 `Remember` 接进 routine，最后用新的顺序让莉莉丝走出这一章。

当前版本还加入了简单开始界面和第二关 `回声台阶`：莉莉丝需要通过平台跳跃拾起三枚回声花瓣，再走向右侧的门。第二关完成后会进入 `Demo 完成` 结尾界面，可按 `Enter` 或 `Space` 回到开始。

## 运行方式

从 Godot 4.7 编辑器打开本目录：

```text
/Users/zhonghaoyang/Code/games/Lilies in Box
```

Godot 打开后运行主场景：

```text
res://scenes/Main.tscn
```

源码级章节自检：

```bash
python3 tools/verify_chapter.py
```

Godot 命令行导入资源并跑游戏 smoke test：

```bash
GODOT_BIN="$HOME/Downloads/Godot.app/Contents/MacOS/Godot"
"$GODOT_BIN" --headless --path . --import
"$GODOT_BIN" --headless --path . --script tools/chapter_smoke_test.gd
```

`chapter_smoke_test.gd` 会实例化第一章和第二关：自动走完第一章三道门，验证 `R` 重开会清空状态回到 `1-1`，再验证第二关三枚回声花瓣会打开终点门，并确认主场景会从第二关完成切到结尾界面。

## 操作

- `A/D` 或左右方向键：移动莉莉丝。
- `W`、上方向键或 `Space`：轻跳。
- `E`：拾取附近积木。
- `Tab`：打开或关闭搭建面板。
- `1/2/3`：在面板打开时把已拥有积木加入 routine。
- `Backspace`：删除 routine 最后一个积木。
- `Enter`：释放朋友执行 routine。
- `R`：重开第一章，方便反复测试三道门。

## Chapter 1：盒子里的三道门

- `1-1 先学会看见`：拾取 `See` 和 `Push`，拼出 `See -> Push`，让朋友把方块推到按钮上。
- `1-2 记忆的重量`：拾取 `Remember`，拼出 `See -> Remember -> Push`，让朋友记住方块和按钮之间的关系。
- `1-3 回声门`：不再给新的积木，要求玩家用已有积木拼出 `Remember -> See -> Push`，完成第一章。

进入每间房时会出现房间标题；打开抽屉时，面板会显示当前房间想听见的 routine。

## Level 2：回声台阶

- 莉莉丝要跳上几段窄平台，拾起三枚回声花瓣。
- 回声花瓣是淡蓝紫色的原创像素碎片，会轻微漂浮和转动。
- 拾齐后右侧高处的门会打开。
- 掉下平台会回到起点，不惩罚，只重新开始这次跳跃。
- 走到门前会进入 `Demo 完成` 结尾界面；按 `Enter` 或 `Space` 可以回到开始。
- 第二关不加入战斗，重点是“她第一次自己跨过空白”的身体感。

## 第一章验收

- 莉莉丝是白发白裙的小女孩形象，有轻微待机呼吸感。
- 房间是横版礼拜堂氛围，不再是纯色块占位。
- 可以拾取 `See`、`Push`、`Remember`，并得到即时文字、发光和轻音效反馈。
- 可以打开搭建面板并按房间要求拼出不同 routine。
- 每间房标题、左上角目标、底部提示和抽屉提示都应指向同一个当前目标。
- 右下角章节状态会显示当前是第几道门，以及已经捡到哪些积木。
- 错误顺序会解释失败原因，流程可以恢复。
- 朋友会执行 routine，把每个房间的方块推到按钮上。
- 前两道门会进入下一房间，最后一道门会触发 Chapter 1 完成提示。
- 第一章完成后可以按 `R` 重开；过门淡出期间按 `R` 会被温柔拦住，避免状态撞车。
- BGM 自动播放；拾取、拼装、错误、朋友出现、开门和章节完成都有低音量反馈，整体安静、不抢戏。
- 开始界面能按 `Enter` 或 `Space` 进入游戏；第一章完成后会进入第二关。
- 第二关必须包含可跳跃平台、三枚可拾取花瓣、开门反馈和完成后的结尾界面。
